CSS Layout

CSS Layouts blir brukt for å kontrollere hvor sidens elementer blir plassert i forhold til dens orginale plassering. De tre mest brukte av disse er CSS Float, CSS Flexbox og CSS Grid.

CSS Float brukes for å bestemme hvor elementer skal "flyte", det betyr at vi dytter elementet dit vi spesifiserer. Nå til dags er det vanligst å bruke float når vi vil at et bilde skal ligge til venstre eller høyre for en tekst. Før vi fikk CSS Flexbox og CSS Grid var det float man brukte for å designe hele websteder, noe som kan være ekremt tungvindt og ikke helt perfekt.

CSS Flexbox (The Flexible Box Module) er det som kom etter float og satt en ny standard innefor CSS Layouts. Flexbox er en endimensjonal layout metode som kan gi jevne mellomrom mellom elementer enten i en row eller column. Ved å bruke "display: flex;" lages det en flex box rundt de elemente det gjelder hvor man ved hjelp av mange froskjellige egenskaper kan bestemme hvor disse elemtene skal ligge inne i flexboxen.
