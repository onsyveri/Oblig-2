Media Typer

I mange tilfeller kan det være nyttig å ta i bruk flere stilark. Dette blir spesielt nyttig når vi gir dem forskjellig betydning. Det gjør vi ved å utnytte egenskapen "media" som settes inn i "link" taggen. Media typene er til for å fortelle nettleseren når den skal bruke de ulike stilarkene.

Selv om det finnes mange forskjellige media typer er det kunn "all", "screen" og "print" man kan være helt sikker på at nettlesere støtter. Fordelene med å bruke media typer er som nevnt at det hjelper nettleseren med å bruke riktig stilark, ulempen med dette er at det er umulig å lage avgrensete medietyper for mobile enheter. Dette er også grunnen til at "handheld" ikke brukes. Det er også mange andre Media typer som ikke brukes lenger fordi de ikke er nødvendig lenger. Som f.eks "projection", "tty" og "tv".
