Git / Github

Git er en teknologi som gir deg en overikt over endringer i et prosjekt. Dette gjør man ved å legge filene sine i et Git repository slik at man kan se endringene man har gjort.

Github er en tjeneste som lar folk bruke git teknologien på tvers av utviklere. Da kan alle se hvem som har gjort hvilke endringer som er ekstremt nyttig under sammarbeidsprosjekter.
